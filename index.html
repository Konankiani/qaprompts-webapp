UI & UX Testing Prompts

Prompts focused on the visual and interactive elements of applications

Generate Test Cases for UI Component

"Act as a senior QA engineer. Generate a comprehensive list of test cases for the following UI component: [Describe the component]. Include positive tests, negative tests, edge cases, and accessibility checks (WCAG 2.1 AA)."

UI Automation Selectors

"I need to write a UI automation script. Provide me with robust CSS and XPath selectors for the following element: [Describe the element]. Give me multiple options and explain which one is most resilient to UI changes."

Accessibility Testing Checklist

"Act as an accessibility testing expert. Create a detailed checklist for testing the [page or feature name] for WCAG 2.1 Level AA compliance. Focus on keyboard navigation, screen reader compatibility, color contrast, and focus indicators."

Cross-Browser Testing Matrix

"Generate a prioritized test matrix for cross-browser and cross-device testing for our new [feature name]. Our target audience uses: 80% Chrome on Windows, 10% Safari on macOS, 5% Safari on iOS, and 5% Chrome on Android. Suggest which tests are critical to run on each platform."

Functional & Business Logic Testing

Prompts focused on verifying application functionality against business requirements

Generate Test Cases from User Story

"Act as a meticulous QA analyst. Here is a user story: [Paste the full user story here]. Create a detailed set of functional test cases, including preconditions, steps, and expected results. Cover all acceptance criteria and think of potential edge cases."

BDD Scenarios (Gherkin Format)

"Write BDD scenarios using Gherkin syntax (Given/When/Then) for the following feature: [Feature name and description]. Include scenarios for the happy path, invalid inputs, and error conditions."

Create Test Data

"I need to test the [feature name]. The form requires: [field specifications]. Generate a table of 10 rows of test data. Include valid data, boundary values, and data that should trigger validation errors for each field."

Explore Edge Cases

"We are developing a [feature name and description]. Brainstorm a list of complex and hidden edge cases we need to test. Think about boundary conditions, concurrent usage, data integrity, and system limits."

API Testing Prompts

Prompts for testing backend services and endpoints

API Test Cases for Endpoint

"Act as a senior API QA engineer. Generate a full suite of test cases for the following REST API endpoint: [Endpoint details, method, purpose, request/response format]. Include contract validation, functionality testing, security, and performance considerations."

Sample API Payloads

"For the [API endpoint] endpoint, generate sample JSON request bodies for the following scenarios: 1. Valid happy path request, 2. Missing required fields, 3. Invalid data formats, 4. Boundary value testing, 5. Unexpected additional fields."

API Test Scripts (Postman)

"Write a JavaScript test script for Postman for the [API endpoint]. The script should verify HTTP status codes, response headers, JSON response structure, and business logic validations. Include error handling for different scenarios."
